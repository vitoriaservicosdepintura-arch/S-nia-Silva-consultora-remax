# Visual Identity: Elite Imobiliária

## Color Palette
- **Primary:** #001A33 (Deep Blue) — background, brand authority
- **Secondary:** #E11B22 (RE/MAX Red) — accents, call to actions, highlights
- **Text:** #FFFFFF (White) — main text, body
- **Muted Text:** #aab8c2 (Light Blue-Gray) — handles, secondary text, metadata

## Typography
- **Headings & Body:** Font Family 'Inter', sans-serif
- **Tweet Text (Hero):** 44px, weight 500
- **Tweet Highlights:** 44px, weight 700, color #E11B22
- **Name:** 38px, weight 700
- **Username:** 28px, weight 400
- **Minimum sizes:** any readable text must be above 20px. Body recommended 34px-44px.

## Layout
- **Viewport:** 1080x1440 px (Instagram Carousel/Portrait Post)
- **Padding:** 80px
- **Grid / Alignment:** Flex column with left-aligned elements inside the main content area.
- **Spacing rules:** 24px gap on headers, 60px margin above main text.

## Composition Rules
- **Logo/profile placement:** Top-left corner, 100x100px rounded circle with 3px white border (`assets/logo.png`).
- **Image treatment:** Bottom-placed large featured image (`assets/hero-bg.jpg`), 500px fixed height, cover sized, with 32px border radius.
- **Visual hierarchy:** Logo & Name -> High-contrast Quote/Hook -> Sub-text -> Visual Element (Property Photo).
- **Style:** "Editorial Twitter snippet" - elegant, minimalistic, focused on authority and a strong hook.

## Adaptation Rules
- When writing shorter messages, the image size can dynamically adapt or flex. However, never use `height: percentage` blindly. Fixed container heights give more predictable outputs in headless browsers.
- Always include the verified checkmark next to the name for brand authority reinforcement.
